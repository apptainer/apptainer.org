apptainer remote remove
-----------------------

Remove an existing apptainer remote endpoint

Synopsis
~~~~~~~~



  The 'remote remove' command allows you to remove an existing remote endpoint 
  from the list of potential endpoints to use.

::

  apptainer remote remove [remove options...] <remote_name>

Examples
~~~~~~~~

::


    $ apptainer remote remove ExampleCloud

Options
~~~~~~~

::

  -g, --global   edit the list of globally configured remote endpoints
  -h, --help     help for remove
