apptainer remote use
--------------------

Set an Apptainer remote endpoint to be actively used

Synopsis
~~~~~~~~



  The 'remote use' command sets the remote to be used by default by any command
  that interacts with Apptainer services.

::

  apptainer remote use [use options...] <remote_name>

Examples
~~~~~~~~

::


    $ apptainer remote use ExampleCloud

Options
~~~~~~~

::

  -e, --exclusive   set the endpoint as exclusive (root user only, imply --global)
  -g, --global      edit the list of globally configured remote endpoints
  -h, --help        help for use
